##   Descripción del Proyecto 
Replicación de la landing page de  Nexcent  según el reto de Frontend Mentor, cumpliendo con los requisitos del Módulo 2 de HTML y CSS.

##  Criterios Cumplidos 

###  HTML Semántico 
- Uso correcto de `<header>`, `<main>`, `<section>`, `<footer>`, `<article>`
- Estructura lógica y accesible
- Comentarios en secciones clave


###  CSS Organizado 
- Archivo externo `styles.css`
- Variables CSS en `:root` para colores y fuentes
- Comentarios por sección
- Código limpio y mantenible

###   Diseño Responsive 
-  3 breakpoints implementados: 
  -  Móvil:  < 768px
  -  Tablet:  768px - 1023px
  -  Escritorio:  ≥ 1024px
- Imágenes responsive

###   Interactividad 
- Efectos hover en botones y enlaces
- Transiciones suaves
- Animaciones opcionales implementadas

###   Organización del Proyecto 
- Estructura de archivos clara
- Assets en carpetas organizadas
- Código comentado
- README completo
